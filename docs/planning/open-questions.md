# Open Questions & Assumptions Log

**When a question resolves,** compress it to a one-line pointer at the doc that now owns the answer (a strategy doc, feature doc, or phase archive) — don't leave long resolved blocks to pile up. This compression is a ritualized phase-close step, not a someday aspiration.

## Format

```

**Context:** Why this question exists. What surfaced it.
**Current thinking:** Where you're leaning, if anywhere. Be honest about uncertainty.
**What would resolve it:** Specific evidence, test, decision, or conversation.
```
